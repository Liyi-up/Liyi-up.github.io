3.Git工作区域

  1. Git Repository(Git 仓库):最终确定的文件保存到仓库，成为一个新的版本，并且对他人可见
  2. 暂存区:暂存已经修改的文件最后统一提交到git仓库中
  3. 工作区(Working Directory)：添加、编辑、修改文件等动作

4.具体添加流程:

  1. git status：查看当前区域的状态
  2. git add xxx：将指定文件添加到暂存区
  3. git commit -m “提交描述”：将暂存区的文件提交到仓库
