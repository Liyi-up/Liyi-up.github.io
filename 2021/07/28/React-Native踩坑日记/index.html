react-native-vector-icons（图标组件）

简介：

react-native-vector-icons组件是react-native开发中，矢量图片的常用解决方案，该组件捆绑了很多图标库，例如：AntDesign、Entypo、Feather、并且该组件能够兼容各个平台终端（Web、IOS、Andorid等）
